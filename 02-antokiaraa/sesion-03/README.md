## Apuntes clase

Raspberry Pi: computadores

Arduino: microcontroladores

en nuestro caso usaremos microcontroladores (Raspberry Pi también tiene)

### Placa a usar: Raspberry Pi Pico 2 W

- desde 5 USD
- inalámbrico: 2.4GHz 802.11n wireless LAN, Bluetooth 5.2
- programable con C, C++ o Python
- chip RP2050.

<https://www.raspberrypi.com/products/raspberry-pi-pico-2/>

otros microcomputadores y microcontroladores:

- BeagleBoard <https://www.beagleboard.org/>
- Electro-Smith <https://electro-smith.com/>
- Bela Platform <https://bela.io/>
